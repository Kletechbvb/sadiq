Optimizing Pipeline Construction

Using Prim's Algorithm to build cost-effective and efficient water supply networks.

The Problem

What is the Problem?

Constructing pipelines to supply water to all areas in a city or region is a costly and complex task. The goal is to connect all areas with pipelines while minimizing the total cost of construction.

Why is it Important?

  • Saves money by avoiding unnecessary construction.
  • Maximizes the reach of pipelines with the available budget.
  • Ensures equitable water distribution.

What Happens When It's Solved?

Solving this problem reduces costs, speeds up project completion, and ensures that everyone gets access to water. It also minimizes environmental impact and promotes sustainable development.

The Algorithm

Prim's Algorithm

Prim's Algorithm is used to find the Minimum Spanning Tree (MST) of a graph. It ensures that all areas are connected with the least total cost.

Steps:

  1. Represent the problem as a graph.
  2. Start with a single node.
  3. Add the cheapest edge connected to the current MST.
  4. Repeat until all nodes are connected.

Why Prim's Algorithm?

  • Efficient and works well for large networks.
  • Guarantees the most cost-effective solution.
  • Easy to implement and understand.

Time and Space Complexity

  • Time Complexity: O(VÂ²) where V is the number of vertices.
  • Space Complexity: O(V) for storing the parent and key arrays.

Prim's Algorithm Implementation

#include iostream
#include vector
#include climits
using namespace std;

// Function to find the vertex with the minimum key value
int findMinKeyVertex(vector& key, vector& inMST, int numVertices) {
    int minKey = INT_MAX; // Initialize minimum key value
    int minIndex = -1;    // Initialize index of the vertex with minimum key

    for (int v = 0; v < numVertices; v++) {
        // Check if the vertex is not in MST and has a smaller key value
        if (!inMST[v] && key[v] < minKey) {
            minKey = key[v];
            minIndex = v;
        }
    }
    return minIndex;
}

// Function to print the constructed MST
void printMST(vector& parent, vector>& graph, int numVertices) {
    cout << "Pipeline Connections:\n";
    cout << "From -> To : Cost\n";
    for (int v = 1; v < numVertices; v++) {
        cout << parent[v] << " -> " << v << " : " << graph[v][parent[v]] << "\n";
    }
}

// Function to construct and find the Minimum Spanning Tree (MST) using Prim's Algorithm
void primMST(vector>& graph, int numVertices) {
    vector parent(numVertices); // Array to store the constructed MST
    vector key(numVertices);    // Key values used to pick the minimum weight edge
    vector inMST(numVertices); // To represent whether a vertex is in MST

    // Initialize all keys as infinite and inMST as false
    for (int i = 0; i < numVertices; i++) {
        key[i] = INT_MAX;
        inMST[i] = false;
    }

    // Start with the first vertex
    key[0] = 0;    // Make key 0 so that this vertex is picked first
    parent[0] = -1; // First node is always the root of MST

    // Construct the MST
    for (int count = 0; count < numVertices - 1; count++) {
        // Pick the vertex with the minimum key value
        int u = findMinKeyVertex(key, inMST, numVertices);

        // Add the picked vertex to the MST
        inMST[u] = true;

        // Update key values and parent index of adjacent vertices
        for (int v = 0; v < numVertices; v++) {
            // Update key only if the edge exists, the vertex is not in MST, and the edge weight is smaller than the current key
            if (graph[u][v] && !inMST[v] && graph[u][v] < key[v]) {
                parent[v] = u;
                key[v] = graph[u][v];
            }
        }
    }

    // Print the constructed MST
    printMST(parent, graph, numVertices);
}

int main() {
    // Example graph representing pipeline costs between areas
    int numVertices = 5; // Number of areas (vertices)
    vector> graph = {
        {0, 2, 0, 6, 0},  // Area 0
        {2, 0, 3, 8, 5},  // Area 1
        {0, 3, 0, 0, 7},  // Area 2
        {6, 8, 0, 0, 9},  // Area 3
        {0, 5, 7, 9, 0}   // Area 4
    };

    // Find and print the MST
    primMST(graph, numVertices);

    return 0;
}
